# Show DeepSeek Models in the Picker

After setting your API key, the DeepSeek models appear in the Copilot Chat model picker:

- **DeepSeek V4-Pro** — flagship 1.6T MoE model (49B active params), 1M context, best for hard coding and math.
- **DeepSeek V4-Flash** — 284B MoE model (13B active params), 1M context, fast and economical.

If the models are hidden, open the Language Models manager and make sure the **DeepSeek** vendor is enabled.

[Open Language Models](command:workbench.action.chat.manage)
